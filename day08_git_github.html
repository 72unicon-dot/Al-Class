Git 필수 명령어

복잡한 명령어는 이제 그만. AI가 다 해주지만, 이 3가지는 알아두면 좋습니다.

git add .
변경된 파일(편지)들을 봉투에 담는다. (Staging)
git commit -m "메시지"
봉투를 밀봉하고 도장을 찍는다. (Save)
git push
우체통(GitHub)에 넣어서 보낸다. (Upload)
