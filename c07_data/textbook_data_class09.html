딥러닝 입문

인공신경망의 원리와 응용

1. 인공신경망(ANN) 원리

인간 뇌를 모방한 인공신경망의 구조와 학습 알고리즘(역전파)을 직관적으로 이해합니다.

2. TensorFlow / Keras

가장 대중적인 딥러닝 프레임워크를 사용하여 간단한 신경망 모델을 직접 구현해봅니다.

3. 모델 학습 및 튜닝

학습률(Learning Rate), 활성화 함수 등 하이퍼파라미터를 조정하여 모델 성능을 최적화합니다.

4. 비정형 데이터 분석 맛보기

이미지 분류 등 비정형 데이터에 딥러닝을 적용하는 사례를 간단한 실습으로 체험합니다.
