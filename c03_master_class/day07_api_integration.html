내 앱에 날개를 달아주는 외부 연결 통로

API(Application Programming Interface)란?

식당의 점원(Waiter)과 같습니다. 손님(Client)이 주문을 하면, 점원이 주방(Server)에 전달하고, 요리가 나오면 손님에게 가져다 줍니다.

자주 쓰는 API 예시

  • 날씨 정보 (OpenWeather)
  • 지도 표시 (Google Maps)
  • 챗봇 대화 (OpenAI GPT)
  • 결제 처리 (Toss Payments)

API Key 관리

보안 주의

API Key는 일종의 비밀번호입니다. GitHub 같은 공개된 저장소에 절대 그대로 올리면 안 됩니다. 해커들이 키를 훔쳐서 과금을 발생시킬 수 있습니다.

Good: 환경 변수(.env) 파일을 사용하여 숨김
Bad: 코드 안에 `const apiKey = "sk-1234..."` 직접 입력
