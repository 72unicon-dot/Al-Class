프론트엔드 vs 백엔드

프론트엔드 (Frontend)

  • 사용자 인터페이스(UI) 디자인
  • 화면 전환, 버튼 클릭 등 상호작용
  • 기술: HTML, CSS, JavaScript, React

백엔드 (Backend)

  • 데이터 저장 및 관리 (DB)
  • 로그인/회원가입 보안 처리
  • 기술: Python(Django/Flask), Node.js, SQL
