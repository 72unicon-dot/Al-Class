코딩 프롬프트 작성법

개떡같이 말해도 찰떡같이 알아듣게 만드는 기술

개발자를 위한 프롬프트 엔지니어링

코딩을 위한 AI 대화는 일반적인 대화와 다릅니다. AI에게 명확한 스펙(Spec)을 주는 것이 핵심입니다.

  • 기술 스택 명시: "Python 3.10과 Pandas 라이브러리를 사용해줘."
  • 입출력 예시 제공: "입력 리스트가 [1, 2, 3]일 때, 출력은 6이 되어야 해."
  • 에러 처리 요구: "빈 리스트가 들어왔을 때 예외 처리 코드를 포함해줘."
Bad Example

피보나치 수열 함수 짜줘

Good Example

Python으로 피보나치 수열을 구하는 함수를 작성해줘. 재귀(Recursion) 방식 말고 메모이제이션(Memoization)을 사용해서 성능을 최적화해줘. 주석으로 각 라인을 설명해줘.

강의 요약 및 다음 단계

자연어로 코드를 작성하며 프로그래밍의 장벽을 낮추고 논리적 사고력을 키웠습니다.
