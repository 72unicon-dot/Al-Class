트리거(Trigger)와 액션(Action)

자동화의 문법: 언제(When) 그리고 무엇을(Do)

자동화 공식: If This, Then That

IF

⚡ 트리거 (Trigger)

자동화를 시작하게 만드는 방아쇠입니다.

예시:

  • 새로운 이메일이 도착했을 때
  • 구글 설문지에 새로운 응답이 제출되었을 때
  • 매주 월요일 오전 9시가 되었을 때 (스케줄)
THEN

🎬 액션 (Action)

트리거가 당겨졌을 때 수행할 행동입니다.

예시:

  • Slack 채널에 메시지 보내기
  • 첨부파일을 드라이브에 저장하기
  • ChatGPT에게 요약을 요청하기

Trigger

Filter

(옵션: 조건 검사)

Action

강의 요약 및 다음 단계

업무 자동화로 효율성을 극대화하는 방법을 익혔습니다.
