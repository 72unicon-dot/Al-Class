학습할 CSS 기법

  • Flexbox: 요소들을 가로/세로로 정렬하는 가장 대중적인 방법
  • Grid: 수평/수직을 동시에 맞추는 격자형 전문 디자인
  • Media Queries: 특정 해상도 이하일 때 디자인 규칙을 바꾸는 핵심 도구
