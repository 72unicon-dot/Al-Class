LECTURE NOTE

IoT 센서 데이터 수집

현장 데이터 수집 및 전처리 실습

1. IoT 센서의 종류

온도, 습도, 진동, 전류 등 산업용 센서의 종류와 데이터 출력 형식을 이해합니다.

2. 데이터 수집 파이프라인

Edge 디바이스에서 Cloud 또는 온프레미스 서버로 데이터를 전송하는 과정을 실습합니다.

3. 결측치 및 노이즈 처리

센서 오류로 인한 결측치와 현장 노이즈를 제거하여 데이터 품질을 높입니다.

4. 데이터 시각화

Grafana 또는 Tableau를 활용하여 실시간 센서 데이터를 모니터링하는 대시보드를 만듭니다.
