Library vs Framework

라이브러리 (Library)

내가 필요할 때 가져다 쓰는 도구 모음. (예: jQuery, React)

주도권: 나(개발자)

프레임워크 (Framework)

이미 갖춰진 뼈대 위에서 규칙에 맞춰 개발. (예: Next.js, Django)

주도권: 프레임워크

컴포넌트(Component)란?

웹사이트를 레고 블록처럼 쪼개서 만드는 방식입니다. 재사용이 가능하여 유지보수가 쉽습니다.
