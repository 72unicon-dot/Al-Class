To-Do 리스트 앱 실습

강의로 돌아가기

내 손으로 만드는 첫 번째 웹앱

리스트에 할 일을 추가하고, 완료 처리하고, 삭제하는 핵심 로직을 AI와 함께 구현합니다.

개발 스택

HTML5 Tailwind CSS JavaScript (ES6)

실습 핵심 포인트

  • 입력창(Input) 데이터 처리
  • 배열(Array)을 이용한 데이터 관리
  • 로컬 스토리지(LocalStorage) 저장
# Cursor에게 지시하기
"간단한 To-Do 리스트 앱을 만들어줘.
할 일을 입력하고 추가 버튼을 누르면 리스트에 추가되어야 해.
삭제 버튼도 있고, 체크박스를 누르면 줄이 그어지게 해줘."
