# Development Rules (개발 규칙)

1. **강의 콘텐츠 보존 (Preserve Lecture Content)**
   - 기존의 강의 보기 내용(텍스트, 커리큘럼 등)은 AI가 임의로 수정하지 않는다.
   - 단순 오탈자 수정 외의 내용 변경은 반드시 승인을 받는다.

2. **관리자 모드 보호 (Protect Admin Mode)**
   - 관리자 모드의 기능은 임의로 수정하지 않는다.
   - 수정이 필요한 경우 반드시 사용자의 승인을 받는다.

3. **백업 필수 (Mandatory Backup)**
   - 기존 파일이나 기능을 수정/삭제할 때는 반드시 `backups` 폴더에 원본을 저장한다.
   - 백업 파일명 형식: `파일명_YYYYMMDD_HHMM.bak` 또는 원본 유지.
