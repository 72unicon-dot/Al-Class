프롬프트 엔지니어링 원리

AI가 완벽하게 이해하는 지시문을 작성하는 핵심 공식

프롬프트의 4대 요소

1. Instruction (지시)

AI가 수행해야 할 구체적인 작업을 명시합니다. "요약해", "번역해", "분석해" 등 동사로 끝나는 명확한 명령어를 사용하세요.

2. Context (맥락)

AI가 상황을 이해할 수 있도록 배경 정보를 제공합니다. "너는 10년차 마케터야", "이 글은 CEO에게 보고할 용도야" 등.

3. Input Data (입력 데이터)

AI가 처리해야 할 실제 데이터입니다. 텍스트, 코드, 데이터 표 등을 명확하게 구분하여 입력합니다.

4. Output Indicator (출력 형식)

원하는 결과물의 형식을 지정합니다. "표로 정리해줘", "불렛포인트 3개로 요약해줘", "JSON 포맷으로 줘" 등.

실전 프롬프트 예시

// 나쁜 예시

이거 마케팅 문구 좀 써줘.

// 좋은 예시

[Role] 너는 전문 뷰티 카피라이터야.

[Task] 20대 여성을 타겟으로 한 새로운 수분 크림의 인스타그램 광고 카피를 작성해줘.

[Context] 제품의 핵심 성분은 히알루론산이고, 끈적임 없이 산뜻한 것이 특징이야.

[Format] 이모지를 포함하여 3가지 버전으로, 각각 해시태그 5개를 포함해서 작성해줘.
