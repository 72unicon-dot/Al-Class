도메인(Domain)이란?

192.168.0.1:3000 my-awesome-app.com

복잡한 IP 주소 대신, 사람이 기억하기 쉬운 영문 주소로 연결해주는 서비스입니다. 가비아, GoDaddy 등에서 구입할 수 있습니다.

HTTPS 보안 접속

HTTP

데이터가 암호화되지 않고 전송됨. 해커가 중간에서 가로챌 수 있음. 브라우저 주소창에 "주의 요함" 표시.

HTTPS

SSL 인증서로 암호화. Vercel 배포 시 무료로 자동 적용됨. 자물쇠 아이콘 표시.
